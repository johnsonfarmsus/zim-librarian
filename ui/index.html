How ZIM Librarian works

What is this?

A private, fully offline research assistant. You give it a library of ZIM files (portable offline copies of Wikipedia, wikis, StackExchange, Project Gutenberg and more), and you can ask questions in plain language. Answers come only from your books — every claim carries a numbered citation you can click to read the original passage, highlighted in the source article.

Adding books

Get .zim files from library.kiwix.org (free; from ~10 MB wikis to all of Wikipedia). Then either drop them into the books folder (shown in the Library tab — picked up automatically, press ⟳ Rescan after adding), or click + Add to pick a file anywhere on disk — the file is used where it is, nothing is copied. Each book is indexed once in the background; big books take a while but only the first time.

Models

Answers are written by a language model that runs entirely on this device. Pick one from the curated list in the Model section and press Download (the one time the app touches the network, and only when you ask). Advanced: any .gguf file placed in the models folder appears in the picker. With no model installed the librarian still works — it returns the most relevant passages verbatim.

Citations you can trust

Sources retrieved for each answer are shown as numbered cards. Numbers like 1 in the answer link to the exact passage. If a sentence has no citation, the sources didn't clearly support it — treat it with suspicion. Language models can make mistakes; the citations exist so you can check.

Chats

Conversations are saved automatically (Chats tab). Follow-up questions understand context — "how do I configure it?" knows what it refers to. Everything is stored locally on this device.

Privacy

No account, no telemetry, no cloud. The app listens only on this device (127.0.0.1) and never sends your questions or documents anywhere. After models and books are downloaded it works with the network fully disconnected, indefinitely.
